# **3.3 Métodos de Strings**

## **a)- Métodos de strings.**
Los métodos de strings son distintas funciones y métodos que nos permiten manipular strings.

# Métodos de Strings
Declaración de un String.
```python
string = 'Bienvenido'

# OUTPUT
'Bienvenido'
```
**Variable de ejemplo**: 
`string = 'hola como te va, a mi bien.'`

Retorna la posición en la que se encuentre la substring. find() retorna la posición de la primera similitud de la substring. Si es substring no es encintrado el método retornará -1especificado.
```python
string.find('como')

# OUTPUT
5
```


Retorna la posición en la que se encuentre la substring. rfind() retorna la última posición de la similitud de la substring. Si es substring no es encintrado el método retornará -1.
```python
string.rfind('hola')

# OUTPUT
-1
```


El método str.join()  es usado para unir varios elementos como items de una lista o diferentes strings. 
```python
print(':'.join(['Python', 'es', 'un lenguaje']))

# OUTPUT
'Python:es:un lenguaje'
```


Concadena ' y ' entre los elementos de la tupla.
```python
print(' y '.join(['A', 'B', 'C']))

# OUTPUT
'A y B y C'
```


Concadena ' ' entre cada substring del string.
```python
print(' '.join('Python'))

# OUTPUT
'P y t h o n'
```


Concadena ', ' entre los elementos de la lista.
```python
lista = ['P', 'y', 't', 'h', 'o', 'n']
coma = ', '
print(coma.join(lista))

# OUTPUT
'P, y, t, h, o, n'
```


El método replace() en su primer parámetro recive el substring que se quiere modificar y como segundo parámetro su reeplazo.
```python
string = 'Hola como te va ?'
nuevo_string  = string.replace('va', 'fue')
print(nuevo_string)

# OUTPUT
'Hola como te fue'
```



El método replace en su primer parámetro recive el substring que se quiere modificar y como segundo parámetro su reeplazo y como tercer parámetro la cantidad de ocurrencias que se quiera modificar.
```python
string = 'Para mi esto es muy bonito'
nuevo_string = string.replace('es', 'ES', 2) 
print(nuevo_string)

# OUTPUT
'Para mi ESto ES muy bonito'
```

## **Estructura para acceder a los substrings de un string**:

variable[inicio:fin]
```python
string = 'Python'
string[2:4]

# OUTPUT
'th'
```

variable[inicio:]
```python
string[1:]  

# OUTPUT
'ython'
```
variable[:fin]
```python
string[:2]  

# OUTPUT
'P'
```

variable[inicio:fin:pasos]
```python
string[0:4:2]

# OUTPUT
'Pt'
```
